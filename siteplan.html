Website Planning Document

Site Name

Explore Philippines - Showcasing the best vacation spots across the Philippines highlighting must-visit locations and travel tips.

Optional domain activity: explorephparadise.com

Site Purpose

This site provides travel guides, destination recommendations, and cultural insights for travelers looking to explore the Philippines. It will include travel itineraries, hotel recommendations, and local attractions.

Scenarios

  • What are the top beach destinations in the Philippines?
  • Which places offer the best adventure activities?

Color Schema

Primary Color: #009688 (Teal- for headers and navigation)

Secondary Color: #FF5722 (Coral Orange - for highlights and buttons)

Typography

Heading Font: Montserrat (Bold for main headings)

Body Font: Roboto (Regular for content text)

Wireframe

Mobile View: A compact layout with a prominent search bar, featured destinations section, and a navigation menu.

Desktop View: A wider layout with a hero image, multiple destination highlights, and detailed travel guides.
